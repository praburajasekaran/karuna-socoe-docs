# Karuna Database Management System

Client: [[Prabodh Jain]]

## Project Overview
Comprehensive database management system for Karuna Club school data with natural language query interface, duplicate prevention, and approval workflows.

## Key Features
- Natural language queries for school searches
- CSV download capability  
- Login/authentication system for access control
- Row limits to prevent bulk data extraction
- PIN code-based duplicate detection
- Manual approval workflow for new registrations
- Data management with entry, edit, and approval processes
- Activity logging for all changes

## Current Status
- Prototype available at: https://demospaces.com/karunadb/
- Database contains 400+ schools in Chennai alone
- 350 PIN codes missing (~12-15% of database)
- 10% database decay over 10 years without updates

## Technical Requirements
- Dual authorization and role-based access control
- Approval workflow with email notifications
- Comprehensive database management with mandatory email addresses
- Advanced search and filtering capabilities
- Bulk import/export capabilities
- Data cleansing for formatting consistency
