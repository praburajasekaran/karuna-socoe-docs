# Book Formatting Guidelines - Kindle & Paperback

## üì± KINDLE FORMATTING (eBook)

### **File Requirements:**
- **Format:** MOBI or EPUB
- **File Size:** Under 650MB
- **Font:** Use standard fonts (Times New Roman, Arial, Calibri)
- **No Custom Fonts:** Amazon will convert to their standard fonts

### **Document Setup:**
- **Page Size:** 8.5" x 11" (standard)
- **Margins:** 1" on all sides
- **Line Spacing:** 1.15 or 1.5
- **Font Size:** 12pt for body text
- **Paragraph Spacing:** 6pt after paragraphs

### **Content Structure:**

#### **Title Page:**
```
[Book Title - Large, Bold, Centered]
[Subtitle - Smaller, Centered]
[Author Name - Centered]
[Publisher/Series Name - Centered]
```

#### **Copyright Page:**
```
Copyright ¬© [Year] [Author Name]
All rights reserved.

ISBN: [Will be assigned by KDP]

No part of this book may be reproduced or transmitted in any form or by any means, electronic or mechanical, including photocopying, recording, or by any information storage and retrieval system, without permission in writing from the author.

[Publisher Information]
[Contact Information]
```

#### **Table of Contents:**
- Use Word's automatic TOC feature
- Include page numbers
- Keep formatting simple and clean

#### **Chapter Headings:**
- **Format:** Heading 1 style
- **Size:** 16-18pt
- **Style:** Bold, Centered
- **Spacing:** 24pt before, 12pt after

#### **Body Text:**
- **Font:** Times New Roman, 12pt
- **Alignment:** Left-aligned (not justified)
- **Indentation:** First line of each paragraph 0.5"
- **No extra spaces** between paragraphs

#### **Special Elements:**
- **Quotes:** Italicized, indented 0.5" from both sides
- **Lists:** Use bullet points or numbers
- **Emphasis:** Bold or italic (not both)
- **Page Breaks:** Insert before each new chapter

### **Kindle-Specific Tips:**
- **No Headers/Footers:** They don't display properly
- **No Page Numbers:** Kindle handles this automatically
- **Simple Formatting:** Avoid complex layouts
- **Test on Kindle:** Preview before publishing

## üìñ PAPERBACK FORMATTING

### **Trim Size Options:**
- **Recommended:** 5.5" x 8.5" (most popular)
- **Alternative:** 6" x 9"
- **Minimum:** 4.25" x 6.87"
- **Maximum:** 8.5" x 11"

### **Page Setup (5.5" x 8.5"):**
- **Page Size:** 5.5" x 8.5"
- **Margins:**
  - **Top:** 0.75"
  - **Bottom:** 0.75"
  - **Inside:** 0.875" (for binding)
  - **Outside:** 0.5"
- **Gutter:** 0.125" (for binding)

### **Content Structure:**

#### **Title Page:**
```
[Book Title - 24pt, Bold, Centered]
[Subtitle - 18pt, Centered]
[Author Name - 16pt, Centered]
[Publisher/Series Name - 14pt, Centered]
```

#### **Copyright Page:**
```
Copyright ¬© [Year] [Author Name]
All rights reserved.

ISBN: [Will be assigned by KDP]

No part of this book may be reproduced or transmitted in any form or by any means, electronic or mechanical, including photocopying, recording, or by any information storage and retrieval system, without permission in writing from the author.

[Publisher Information]
[Contact Information]
```

#### **Table of Contents:**
- **Font:** Times New Roman, 12pt
- **Format:** Left-aligned with dot leaders
- **Page Numbers:** Right-aligned

#### **Chapter Headings:**
- **Format:** Heading 1 style
- **Size:** 18pt
- **Style:** Bold, Centered
- **Spacing:** 36pt before, 18pt after

#### **Body Text:**
- **Font:** Times New Roman, 11pt
- **Alignment:** Justified (both left and right)
- **Indentation:** First line of each paragraph 0.5"
- **Line Spacing:** Single (1.0)
- **Paragraph Spacing:** 6pt after paragraphs

#### **Headers and Footers:**
- **Header:** Book title (left) or chapter title (right)
- **Footer:** Page numbers (centered)
- **Font:** Times New Roman, 10pt

### **Paperback-Specific Requirements:**
- **Page Count:** Must be even number (add blank page if needed)
- **Bleed:** Not required for text-only books
- **Cover:** Separate file (front, back, spine)
- **Binding:** Perfect bound (glued)

## üõ†Ô∏è FORMATTING TOOLS & SOFTWARE

### **Recommended Software:**
1. **Microsoft Word** (most common)
2. **Google Docs** (free alternative)
3. **Scrivener** (for complex projects)
4. **Vellum** (Mac only, professional)

### **Word Formatting Steps:**

#### **For Kindle:**
1. Create new document
2. Set page size to 8.5" x 11"
3. Set margins to 1" all around
4. Format headings as Heading 1, 2, 3
5. Set body text to Times New Roman, 12pt
6. Insert page breaks before chapters
7. Save as .docx file
8. Convert to MOBI using Calibre or KDP

#### **For Paperback:**
1. Create new document
2. Set page size to 5.5" x 8.5"
3. Set margins as specified above
4. Format headings and body text
5. Add headers and footers
6. Insert page numbers
7. Save as .docx file
8. Upload directly to KDP

## ‚ö†Ô∏è COMMON FORMATTING MISTAKES TO AVOID

1. **Don't use:** Custom fonts, complex layouts, headers/footers in Kindle
2. **Don't use:** Justified text in Kindle (use left-aligned)
3. **Don't use:** Page numbers in Kindle (automatic)
4. **Don't use:** Odd number of pages in paperback
5. **Don't use:** Margins too small (text will be cut off)
6. **Don't use:** Font sizes too small (hard to read)
